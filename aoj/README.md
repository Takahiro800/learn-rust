# Aizu Online JudgeをRustで埋める

## 毎回の作業
1. fileを作る
   1. `touch src/**.rs`
2. Cargo.tomlに設定を加える
   1. snipet追加済
3. inputをマクロを使えないので、alfredからmacroを貼り付ける(aoj)
4. テストする
   1. `cargo run --bin (binのname)`
